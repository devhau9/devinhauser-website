# media/ – Rohmaterial für die Website

Hier werden alle Bilder, Videos, Logos und Dokumente abgelegt, bevor sie
in die Website eingebunden werden.

- photos/iqfoil — IQFoil-Actionbilder
- photos/wingfoil — Wingfoil-Racing-Actionbilder
- photos/portrait — Portraitfotos
- photos/lifestyle — Lifestyle-/Behind-the-Scenes-Bilder
- videos/reels — kurze Reels/Social-Content
- videos/highlights — längere Highlight-/Wettkampfvideos
- logos/partners — Partnerlogos (nur mit bestätigter Freigabe verwenden)
- logos/devin-hauser — eigenes Logo/Wortmarke, falls vorhanden
- documents/sponsoring-dossier — Sponsoring-Dossier als PDF

Wichtig: Bildcredits pro Datei erfassen (z. B. im Dateinamen oder in einer
separaten Credits-Liste), bevor Bilder auf der Website verwendet werden.

Alle Ordner sind aktuell leer und warten auf Befüllung durch Devin bzw.
Sichtung der Dateien im SSD-Bereich.
