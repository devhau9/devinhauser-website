# Design-System (Entwurf, noch nicht final)

## Stilrichtung
Modern, minimalistisch, hochwertig, sportlich, international.
Inspiration: Red Bull Athleten-Websites, On, Apple, Nike, moderne Athleten-Websites.

## Prinzipien
- Grosse, hochwertige Bilder und Videos als Haupt-Gestaltungsmittel
- Viel Weissraum, klare Hierarchie
- Hochwertige, aber zurückhaltende Animationen (kein Overdesign)
- Mobile First, Desktop-Erlebnis gleichwertig hochwertig
- Schnelle Ladezeit als Designprinzip (Bild-/Videokompression von Anfang an mitdenken)

## Farbpalette (Vorschlag, zu bestätigen)
- Schwarz / sehr dunkles Blau (Basis)
- Weiss (Basis)
- Rot (Akzentfarbe)
- Hellgrau (sekundär)
